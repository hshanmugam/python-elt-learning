# Topic 13 Mini Quiz

1. What does a test assertion do?
   Answer: It checks that an actual result matches an expected result
2. Why test null handling?
   Answer: Because missing data is common in pipelines
3. What is a fixture?
   Answer: Reusable setup for tests
4. Why are tests valuable before orchestration?
   Answer: They catch logic problems before repeated scheduled runs
5. What is one popular Python testing framework?
   Answer: `pytest`
