# Topic 1 Mini Quiz

1. Which Python type is best for a customer name?
   Answer: `str`
2. Which Python type is best for a row count?
   Answer: `int`
3. What does `None` usually represent?
   Answer: A missing or unavailable value
4. Which function converts `"25"` to an integer?
   Answer: `int()`
5. Why are f-strings useful in ETL scripts?
   Answer: They create readable log and status messages
