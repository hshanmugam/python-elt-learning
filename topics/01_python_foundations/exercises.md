# Topic 1 Exercises

1. Create variables for `pipeline_name`, `source_system`, and `batch_id`.
2. Store the string `"150"` in a variable and convert it to an integer.
3. Store the string `"89.95"` in a variable and convert it to a float.
4. Create a boolean variable named `is_file_ready`.
5. Write an f-string that says a pipeline loaded 500 rows.
6. Add two integer variables named `success_count` and `failure_count` and store their total.
7. Compare whether `row_count` is greater than 1000.
8. Create a variable with value `None` to represent a missing email address.
9. Convert the integer `25` to a string and place it in a message.
10. Write a small script that converts `"42"` and `"18.5"` into numeric types and prints both.
11. Create variables for `raw_discount = "0.10"` and convert it to a float.
12. Write a status message using `source_name`, `rows_read`, and `rows_loaded`.
