# Topic 5 Mini Quiz

1. Which module is commonly used for CSV files?
   Answer: `csv`
2. Which module is commonly used for JSON?
   Answer: `json`
3. What does `Path` help with?
   Answer: Clean and reliable path handling
4. Why is `DictReader` useful?
   Answer: It reads CSV rows as dictionaries
5. Why is file handling central to ETL?
   Answer: Many pipelines ingest and publish data through files
