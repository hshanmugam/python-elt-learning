# Topic 5: Files and Serialization

## Core ideas

ETL and ELT workflows often start and end with files. Python makes it easy to read and write text, CSV, and JSON.

Key tools:
- `open()` for text files
- `csv` for flat files
- `json` for semi-structured data
- `pathlib.Path` for path handling

You will read source files, stage outputs, capture rejected records, and write transformed data. File handling is one of the most practical Python skills for data engineering.
