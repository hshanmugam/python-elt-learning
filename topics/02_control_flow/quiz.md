# Topic 2 Mini Quiz

1. Which statement is used for branching?
   Answer: `if`
2. Which loop is common for iterating through records?
   Answer: `for`
3. What does `continue` do?
   Answer: It skips the current loop item
4. What does `break` do?
   Answer: It exits the loop
5. Why are comprehensions useful in ETL scripts?
   Answer: They make filtering and transformation concise
