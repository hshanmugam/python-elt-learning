# Topic 2 Exercises

1. Write an `if` statement that checks whether `rows_loaded` is 0.
2. Use `if` and `else` to label a record as valid or invalid based on `amount >= 0`.
3. Loop through a list of file names and print each one.
4. Loop through numbers 1 to 5 and print them.
5. Use `continue` to skip negative values in a list.
6. Use `break` to stop a loop when a value equals `"STOP"`.
7. Create a list comprehension that doubles each number in `[1, 2, 3, 4]`.
8. Create a list comprehension that keeps only even numbers from 1 to 10.
9. Write a `while` loop that counts from 1 to 3.
10. Loop through records and count how many have `is_valid = True`.
11. Write logic that labels totals above 1000 as `"high_value"`, otherwise `"standard"`.
12. Loop through rows and print only rows where `country == "USA"`.
