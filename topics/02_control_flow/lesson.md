# Topic 2: Control Flow

## Core ideas

Control flow decides what your program does next. In pipelines, this means deciding which records pass validation, how batches are processed, and what happens when a condition is met.

## Key tools

- `if`, `elif`, and `else` for branching
- `for` and `while` loops for repeated work
- `break` and `continue` for loop control
- comprehensions for compact filtering and transformation

Control flow is used for row validation, duplicate filtering, pagination, threshold checks, and retry loops.
