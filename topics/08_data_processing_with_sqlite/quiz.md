# Topic 8 Mini Quiz

1. Which built-in module connects Python to SQLite?
   Answer: `sqlite3`
2. What SQL statement creates a table?
   Answer: `CREATE TABLE`
3. What method saves inserts permanently?
   Answer: `commit()`
4. In ELT, when does most transformation happen?
   Answer: After loading data into the target system
5. Why is SQLite useful here?
   Answer: It provides a simple database environment for learning pipeline loading patterns
