# Topic 8: Data Processing with SQLite

## Core ideas

SQLite is a lightweight relational database built into Python through `sqlite3`. It is excellent for learning pipeline loading patterns without installing a full database server.

Important ideas:
- connecting to SQLite
- creating tables
- inserting rows
- querying results
- ETL versus ELT workflow design

Many real workflows move between Python and SQL. Understanding that boundary is essential for data engineering work.
