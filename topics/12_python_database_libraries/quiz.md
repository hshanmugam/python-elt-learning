# Topic 12 Mini Quiz

1. Why should SQL values be parameterized?
   Answer: To keep SQL safer and more reliable
2. What saves a transaction?
   Answer: `commit()`
3. What object commonly executes SQL?
   Answer: A cursor
4. What is one popular higher-level Python DB toolkit?
   Answer: SQLAlchemy
5. Why do database libraries matter in ETL?
   Answer: They connect Python pipeline code to relational targets
