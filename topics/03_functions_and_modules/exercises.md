# Topic 3 Exercises

1. Write a function that adds two numbers.
2. Write a function named `normalize_name` that strips spaces and applies title case.
3. Write a function that returns `True` if an amount is non-negative.
4. Write a function with two parameters: `quantity` and `price`.
5. Write a function that returns a dictionary with keys `customer_id` and `email`.
6. Write a function that converts a list of names to lowercase.
7. Create a file-level constant called `PIPELINE_NAME`.
8. Write a function that prints a status message for rows loaded.
9. Write a function that receives a record and returns only the `customer_id`.
10. Write a function called `reject_record` that returns a reason string.
11. Explain when you would split code into more than one module.
12. Write a function that defaults `country` to `"USA"` if not provided.
