# Topic 3 Mini Quiz

1. What keyword starts a function definition?
   Answer: `def`
2. What does a return statement do?
   Answer: It sends a value back from the function
3. Why are small functions helpful in ETL?
   Answer: They are easier to test and reuse
4. What is a module?
   Answer: A Python file containing code you can import
5. Name three common ETL module categories.
   Answer: Extract, transform, and load
