# Topic 11 Mini Quiz

1. Which HTTP method is common for data extraction?
   Answer: `GET`
2. What format do many APIs return?
   Answer: JSON
3. Why do pipelines need pagination logic?
   Answer: Because APIs often return large datasets in pages
4. Why is a timeout useful?
   Answer: It prevents jobs from waiting forever
5. Why store raw API responses?
   Answer: For traceability, replay, and debugging
