# Topic 6 Mini Quiz

1. Which block catches an error?
   Answer: `except`
2. Which block always runs at the end?
   Answer: `finally`
3. Which module is commonly used for logs?
   Answer: `logging`
4. Which log level is more serious: `INFO` or `ERROR`?
   Answer: `ERROR`
5. Why does error handling matter in ETL?
   Answer: It prevents one bad record or conversion from breaking the whole job unnecessarily
