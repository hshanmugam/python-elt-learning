# Topic 4 Exercises

1. Create a list containing three file names.
2. Create a dictionary for a customer record with `customer_id`, `name`, and `country`.
3. Access the `country` value from a dictionary.
4. Add a new key `status` to a dictionary.
5. Create a tuple containing `("orders.csv", "sales_orders")`.
6. Create a set with duplicate ids and observe that duplicates are removed.
7. Loop through a list of dictionaries and print each `customer_id`.
8. Use a dictionary to count how many times each country appears.
9. Create a nested dictionary with a customer and one order.
10. Write code that checks whether a key `email` exists in a record.
11. Append a new record to a list.
12. Remove duplicates from a list of ids using a set.
