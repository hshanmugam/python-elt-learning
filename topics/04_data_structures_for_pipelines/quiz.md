# Topic 4 Mini Quiz

1. Which structure is best for key-value row data?
   Answer: `dict`
2. Which structure automatically keeps only unique values?
   Answer: `set`
3. Which structure is ordered and mutable?
   Answer: `list`
4. Which structure is ordered and immutable?
   Answer: `tuple`
5. Why are dictionaries common in ETL?
   Answer: They represent records naturally as column-name to value mappings
