# Topic 9 Mini Quiz

1. What alias is commonly used for NumPy?
   Answer: `np`
2. What is a vectorized operation?
   Answer: An operation applied to an entire array at once
3. What does a boolean mask contain?
   Answer: `True` and `False` values used for filtering
4. Which function returns an average?
   Answer: `np.mean`
5. Why is NumPy relevant to data engineering?
   Answer: It supports efficient numeric processing and column-style transformations
