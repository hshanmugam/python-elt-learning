# Topic 10: Pandas for ETL

## Core ideas

Pandas provides the `DataFrame`, a table-like structure that is extremely useful for batch data preparation, cleanup, aggregation, and joining.

## Why it matters for ETL and ELT

- It reads CSV, Excel, JSON, and SQL data into a common structure
- It makes column transformations concise
- It supports filtering, joins, groupings, and missing value handling

## Key concepts

- `Series` and `DataFrame`
- selecting rows and columns
- filtering
- derived columns
- joins and merges
- group by and aggregations
- null handling

## Pipeline examples

- standardize column names
- calculate revenue columns
- join customer and order data
- produce aggregated outputs before loading or reporting
