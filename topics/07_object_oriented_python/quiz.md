# Topic 7 Mini Quiz

1. What does `self` refer to?
   Answer: The current object instance
2. What decorator creates a dataclass?
   Answer: `@dataclass`
3. Why are dataclasses useful in ETL?
   Answer: They model records cleanly with less boilerplate
4. What is a method?
   Answer: A function defined inside a class
5. Why can classes help in pipeline projects?
   Answer: They organize state and behavior together
