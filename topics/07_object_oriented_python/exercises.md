2. Add an `__init__` method that stores `name`.
3. Add a method named `run` that prints a message.
4. Create an object from your `PipelineJob` class.
5. Create a dataclass named `CustomerRecord`.
6. Add fields `customer_id`, `name`, and `country` to the dataclass.
7. Add a method to check whether `country` equals `"USA"`.
8. Explain why a dataclass is useful for row-like objects.
9. Create a class named `Validator` with a method `is_valid_amount`.
10. Instantiate `Validator` and test it.
11. Add a method that returns a readable summary string.
12. Explain one advantage of classes in larger ETL scripts.
