# Python for ELT and ETL Learning Repository

This repository is a hands-on learning path for mastering Python concepts used in ELT and ETL pipelines.

## What you will find here

- A roadmap and study plan focused on pipeline engineering
- Separate topic folders for each Python concept
- Explanations with ETL and ELT examples
- Example Python files with comments
- Ten or more exercises per topic
- Exercise answers for self-checking
- Mini quizzes for quick review
- Mini projects based on realistic company data scenarios

## Recommended study order

1. [Roadmap](C:\Projects\Etl-Elt\LearnPython\docs\roadmap.md)
2. [Study Plan](C:\Projects\Etl-Elt\LearnPython\docs\study_plan.md)
3. Topic 1 through Topic 8 in order
4. Mini projects after completing the related topics

## Repository structure

- `docs/` contains the learning roadmap and plan
- `topics/` contains concept-by-concept lessons and practice
- `projects/` contains realistic ETL and ELT mini projects
