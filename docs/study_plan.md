# Study Plan

## 8-Week Plan

### Week 1

- Topic 1: Python Foundations
- Topic 2: Control Flow
- Outcome: write simple scripts that clean and inspect records

### Week 2

- Topic 3: Functions and Modules
- Topic 4: Data Structures for Pipelines
- Outcome: organize code into reusable pipeline steps

### Week 3

- Topic 5: Files and Serialization
- Outcome: read and write CSV and JSON confidently

### Week 4

- Topic 6: Error Handling and Logging
- Outcome: build scripts that can be monitored and debugged

### Week 5

- Topic 7: Object-Oriented Python
- Outcome: model pipeline jobs and records cleanly

### Week 6

- Topic 8: Data Processing with SQLite
- Outcome: combine Python processing with SQL tables

### Week 7

- Project 1 and Project 2
- Outcome: practice ETL and ELT patterns end to end

### Week 8

- Project 3
- Review all quizzes and repeat weak topics
- Outcome: finish with a small portfolio of data engineering practice work
